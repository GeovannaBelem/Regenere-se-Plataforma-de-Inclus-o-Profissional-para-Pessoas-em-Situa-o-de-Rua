# Regenere-se

**Regenere-se** é uma plataforma web que conecta pessoas em situação de rua a oportunidades de emprego, por meio da triagem de currículos e integração com ONGs e empresas. Desenvolvida com HTML, CSS, JavaScript, PHP e Figma, a solução visa promover inclusão social com tecnologia.

## 🛠 Tecnologias Utilizadas

- HTML5  
- CSS3  
- JavaScript  
- PHP  
- Figma (protótipos e UI/UX)

## 🎯 Funcionalidades

- Cadastro de candidatos
- Triagem curricular automatizada
- Comunicação entre ONGs e empresas
- Plataforma acessível e intuitiva

## 🎓 Contexto Acadêmico

Este projeto foi desenvolvido como parte de uma entrega final para a banca acadêmica, com foco em impacto social e uso de tecnologia para inclusão.

## 🤝 Contribuições Futuras

- Integração com APIs de empregos
- Dashboard para ONGs e empresas
- Melhorias em acessibilidade e responsividade
